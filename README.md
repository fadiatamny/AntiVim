# Anti Vim
Anti-Vim is a text editor for programmers inspired by the [ded](https://github.com/tsoding/ded) project.
Watching along and coding as we go in my own flavour mainly with c++ instead of C.

### Dependencies
- SDL2

### How to build
```make build```

### How to run 
```make run```
